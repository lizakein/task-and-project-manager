# 🗂️ Task & Project Manager

Мой пет-проект для практики работы с **React**, **TypeScript** и **Zustand**.  
Это менеджер задач и проектов, вдохновлённый популярными инструментами вроде Trello и Asana.  
Приложение помогает планировать проекты, управлять задачами, приоритетами и статусами — с удобным интерфейсом и drag & drop функциональностью.

---

## ✨ Функциональность

### 🔹 Проекты и задачи

- Создание, редактирование и удаление проектов и задач
- Каждая задача имеет:
  - название и описание
  - приоритет (низкий, средний, высокий)
  - теги (цветовые метки)
  - статус (To Do / In Progress / Done)
  - дедлайн выполнения
- Перемещение задач между статусами с помощью drag & drop
- Предупреждающие модальные окна при удалении или выходе без сохранения
- Глобальный поиск задач и проектов

### 🔹 Фильтры, сортировка и теги

- Фильтрация задач по тегам и приоритету
- Сортировка задач по названию, приоритету и дате
- Возможность добавлять, изменять и удалять цветовые метки
- Цвет тега обновляется глобально при изменении

### 🔹 Пользовательский интерфейс

- Адаптивная верстка (desktop-first, гамбургер-меню для навигации)
- Доступность (ARIA-атрибуты, управление с клавиатуры)
- Поддержка нескольких страниц с помощью React Router
- Дизайн частично вдохновлен макетом из Figma, остальное продумано и реализовано мной

### 🔹 Состояние и данные

- Управление состоянием через Zustand
- Типизация данных и компонентов с помощью TypeScript
- Сохранение данных в localStorage (задачи и проекты не теряются при перезагрузке страницы)

---

## 🧩 Технологии

| Технология       | Использование                                          |
| ---------------- | ------------------------------------------------------ |
| **React (Vite)** | Компонентная архитектура и модульная структура         |
| **TypeScript**   | Статическая типизация, интерфейсы, строгие типы данных |
| **Zustand**      | Глобальное управление состоянием                       |
| **React Router** | Многостраничная навигация                              |
| **React DnD**    | Drag & drop задач между статусами                      |
| **localStorage** | Сохранение проектов и задач между сессиями             |
| **CSS3**         | Адаптивная стилизация без фреймворков                  |

## ⚙️ Установка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/lizakein/task-and-project-manager.git

# Перейти в папку проекта
cd task-and-project-manager

# Установить зависимости
npm install

# Запустить локально
npm run dev

# Открыть в браузере
http://localhost:5173
```
